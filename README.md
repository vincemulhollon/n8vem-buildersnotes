n8vem-buildersnotes
============

Docbook builders notes for N8VEM boards

These builders notes were developed on Debian Linux using the following packages:
docbook5-xml
docbook-xsl-ns
xsltproc
fop
xmlto
libxml2-utils
xmlstarlet
