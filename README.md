n8vem-manual
============

Docbook manuals for N8VEM boards

These manuals were developed on Debian Linux using the following packages:
docbook5-xml
docbook-xsl-ns
xsltproc
fop
xmlto
libxml2-utils
xmlstarlet

Authored by the N8VEM community
